# AfterEffects-EasyText
EasyText是一款专为 Adobe After Effects 设计的高效文字层管理脚本。它能够深度扫描全项目中的所有文字层，并提供直观的列表界面进行统一预览和属性批量修改。EasyText is a powerful management and batch editing script for Adobe After Effects. It streamlines the process of finding and modifying text layers across complex projects through an intuitive list-based interface.
