# Easy Text v1.0
Easy Text is a powerful management and batch editing script for Adobe After Effects. It streamlines the process of finding and modifying text layers across complex projects through an intuitive list-based interface.

âœ¨ Key Features
ğŸ” Project-wide Deep Scan: Recursively scans all compositions and nested comps to find every text layer in your project.

âš¡ Batch Property Editing: Modify Font, Text Content, Scale, Size, Color, Stroke, Tracking, and Leading for multiple layers simultaneously.

ğŸ“ Intelligent Leading: The leading field natively supports "auto" input to quickly restore After Effects' default auto-leading state.

â­ Font Favorites: Save your most-used fonts to a favorites list that persists across sessions.

ğŸ¯ Quick Navigation: Double-click any row in the list to instantly open the corresponding composition and select the target layer.

ğŸ›¡ï¸ Safe Management: Remove items from the list using the Delete/Backspace key without affecting the actual layers in your project.

Download the EasyText.jsx file.

Move it to your After Effects directory:

Windows: Program Files\Adobe\Adobe After Effects <version>\Support Files\Scripts\ScriptUI Panels

macOS: /Applications/Adobe After Effects <version>/Scripts/ScriptUI Panels

Restart After Effects.

Find EasyText.jsx under the Window menu.

ğŸš€ How to Use
Scan: Click "Deep Scan Project Text Layers" to populate the list.

Select: Click a row to select it, or use Ctrl/Cmd + Click for multiple selections.

Edit: Enter new values in the "Property Editor" panel at the bottom.

Apply: Use "Apply Sel" to update selected layers, or "Apply All" to overwrite every text layer in the list.
